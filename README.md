 #동영상 시청 사이트
 html과 css를 이용해 동영상 시청을 위한 테스트 사이트 구성
 
 ## 특징
 - 고정 페이지 : 너비 1440px
 
 ### 파일구조
 ```
 │-- img			이미지 파일
 ├ css				css 파일
 ├ index.html		메인페이지
 ├ header.html		header
 ├ footer.html		footer
 ├ search.html      검색창
 ├ my_room.html     개인 컬렉션
 ├ login.html       로그인 페이지
 ├ subcribe.html    가입페이지
 └ family.html      선택영상 상세 페이지
 ```
 ### 사용 기술
 html, css
